Welcome To simpleGrid!

simpleGrid is a customizeable 12-column grid system written entirely in CSS. simpleGrid is responsive, and works great with mobile devices. The goal of this project is to have a fast, easy to setup grid without a bunch of fluff and unnecessary widgets. Currently, simpleGrid only supports Desktops and Mobile Phones; native Tablet support is planned for the future.

Desktop Grid System

Use these classes in your Divs to use the desktop grid system. By default, the desktop grid system will collapse into a single column on mobile devices. Use the desktop grid system in conjunction with the mobile grid for easy responsive layouts.
